# Sycom LMS Sitemap

This document describes the structure and content sections for each page in the Sycom Learning Management System, focusing on logged-out pages.

## Home (`/`)

- **Navbar**
  - Logo and branding
  - Navigation links (Blog, Courses, Pathway)
  - Phone number
  - Sign in button

- **Hero Header Section**
  - Engaging introduction to the cyber security learning platform
  - Highlights value proposition and mission
  - Primary call-to-action button

- **Features List Section**
  - Overview of up to three key platform features
  - Examples: interactive labs, expert instructors, certification paths
  - Feature cards with icons and descriptions

- **Featured Courses List Header Section**
  - Section title and description
  - Highlights selected courses or popular learning tracks
  - Grid/list of featured course cards

- **Testimonial Section**
  - Quotes and feedback from successful students
  - Testimonials from industry professionals
  - Student success stories

- **CTA Section (Call To Action)**
  - Strong call to action encouraging visitors to sign up
  - Encourages exploration of courses
  - Contact information for more information

- **FAQ Section (Frequently Asked Questions)**
  - Answers to frequently asked questions about the platform
  - Questions about course structure
  - Support and enrollment information

- **Footer**
  - About Sycom Academy
  - Popular Courses links
  - Recent Posts links
  - Contact details (address, phone, email)
  - Social media links
  - Copyright information
